# Ruby/GOffice

Ruby/GOffice is a Ruby binding of GOFFICE which is needed by GOffice.

## Requirements

* Ruby/GTK3 and Ruby/GSF in
  [Ruby-GNOME2](http://ruby-gnome2.sourceforge.jp/)
* [GOffice](https://developer.gnome.org/goffice/)

## Install

    gem install goffice
