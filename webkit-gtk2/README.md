# Ruby/WebKitGTK2

Ruby/WebKitGTK2 is a Ruby binding of WebKitGTK+ for GTK+ 2.

## Requirements

* Ruby/GObjectIntrospection and Ruby/GTK2 in
  [Ruby-GNOME2](http://ruby-gnome2.sourceforge.jp/)
* [WebKitGTK+](http://webkitgtk.org/)

## Install

    gem install webkit-gtk2
