# Ruby/ClutterGTK

Ruby/ClutterGDK is a Ruby binding of Clutter-GDK.

## Requirements

* Ruby/Clutter and Ruby/GTK3 in
  [Ruby-GNOME2](http://ruby-gnome2.sourceforge.jp/)
* [Clutter-gdk](http://blogs.gnome.org/clutter/)

## Install

    gem install clutter-gdk
